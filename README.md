- ⚡ **Async Ultra-Fast Port Scanning**

- 🔑 **Banner & TLS Certificate Grabber**
- 🛰 **Reverse DNS Lookup**
- 📊 **Reports in JSON + HTML formats**
- 🚫 **Rootless** — runs without admin privileges
- 💻 Cross-platform: **Linux | Windows | Mac**
